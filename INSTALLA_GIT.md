Apri PowerShell e digita:
```powershell
git --version
```

Dovresti vedere qualcosa come `git version 2.x.x`

## Prossimi Passi

Dopo aver installato Git, potrai:
1. Inizializzare il repository: `git init`
2. Aggiungere i file: `git add .`
3. Fare commit: `git commit -m "messaggio"`
4. Caricare su GitHub
